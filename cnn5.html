Interactive multi-level CNN for binary line detection

This demo trains a CNN on binary images whose size can range from 5×5 to 32×32. A pixel value of 1 means black. The default target is 1 when the image contains a complete black row, column, or main diagonal across the full image. The model is configurable: input size, convolution filter size, number of convolution levels, filters per level, dense hidden width, batch size, learning rate, and training speed are all adjustable.

1. Generated dataset

Balanced positives and near-miss negatives. Click a thumbnail to inspect it.

2. Target rule

Class 1 means a complete black row, column, or main diagonal across the whole image.

3. Editable input image

Click cells to toggle pixels. For large images, use the buttons to generate examples.

4. Full CNN pipeline

Input → convolution levels → feature maps → flattening → dense projection → sigmoid answer.

5. Learned filters

Click a filter tile. For deeper levels, each tile aggregates all input-channel kernels.

6. Feature maps by CNN level

Each convolution level produces smaller feature maps that are passed to the next level.

7. Local convolution computation

One output cell is a dot product between a local patch and a filter, plus bias, then ReLU.

8. Flattened vector → projection → output

The final feature maps are flattened into a vector, optionally projected to a dense hidden layer, then mapped to one logit.

9. Backpropagation view

Signed gradients show which input pixels and selected filter weights would most change the loss.

10. Training curve

Use lower speed, smaller batches, SGD, fewer filters, or smaller dense width to slow convergence.

Interpretation: a convolution level applies several K×K filters at every valid location. This creates feature maps with spatial size S − K + 1. Stacking levels means that later filters do not see raw pixels directly; they see lower-dimensional feature maps produced by earlier filters. After the last level, all feature maps are flattened into one vector. A dense projection converts that vector into either a hidden representation or directly into a single logit, and the sigmoid turns the logit into P(class = 1).
