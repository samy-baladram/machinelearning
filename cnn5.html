Interactive CNN classifier for 5×5 binary images

The input is a 5×5 binary image. A pixel value of 1 means black. The target is 1 when the image contains a complete line of five black pixels: one full row, one full column, or one of the two full diagonals. The visualization is designed to make the pipeline explicit: convolution filters create feature maps, feature maps are ReLUed and flattened into a vector, then a dense projection maps that vector to a binary prediction.

4. Learned filters

Each kernel is a local template. Positive weights are orange, negative weights are blue.

5. Feature maps

Each filter slides across the 5×5 image and creates one activation map after ReLU.

6. Flattened vector → projection

The ReLU feature maps are flattened into a vector. A dense layer projects that vector toward the final answer.

7. Full CNN network representation

This view shows the exact shape transition: 5×5 input → filters → feature maps → flattened vector → dense projection → sigmoid probability.

8. Backpropagation

Gradients show how the selected image and selected filter would change to reduce the current loss.

9. Training curve

Defaults are intentionally slower than the 3×3 version. Increase learning rate, filters, width, or train speed to accelerate convergence.

Architecture: 5×5 input → F valid convolution filters of size K×K → ReLU feature maps of size (6−K)×(6−K) → flatten into a vector of length F(6−K)² → optional dense hidden layer → sigmoid output. The target is a full row, full column, or full diagonal of five black pixels. The default setting uses smaller learning rate and fewer updates per frame so convergence is visible rather than immediate.
